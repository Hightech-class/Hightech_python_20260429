### 자료구조
* 컴퓨터에서 자료를 정리하고 조직화하는 여러가지 구조
    * 자료마다 효율적인 정리 규칙이 존재
    * 단순 자료 구조(숫자, 문자)
    * 복합 자료 구조(여러 자료들을 한꺼번에 보관함(컨테이너))

* 복합 자료 구조
    *  선형(linear) 자료구조 : (자료들을 일렬로 나열(순서) / ex. 리스트, 스택, 큐, 덱)
    *  비선형(non-linear) 자료구조 : (한줄로 나열하기 어려운 자료들 / ex. 트리, 그래프)
